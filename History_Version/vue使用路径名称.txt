使用路径名称

在定义路径规则的时候，如果你给它提供了一个name属性，则可以在后续使用这条路径规则的时候，直接引用。

router.map({
  '/user/:userId': {
       name: 'user',
    component: {...}
  }
});
在v-link中使用

<a v-link="{name: 'user', params: {userId: 1}">This is a user whose id is 1</a>
还可以使用router.go()

router.go({name: 'user', params: {userId: 1}});
最终都会匹配到/user/1这条路径上
